# Release v0.0.15 - Password Management Integration

**Release Date:** December 30, 2024  
**Build:** 15

## 🎉 New Feature: Password Management

### Password Manager Screen
- ✅ **New navigation tab** - "Passwords" with key icon
- ✅ View all saved passwords (keys visible, values encrypted)
- ✅ Search/filter passwords by key name
- ✅ Add new passwords with encryption
- ✅ Delete passwords with confirmation
- ✅ Beautiful empty state UI

### Add Password Dialog
- ✅ Key name input (unique validation)
- ✅ Password input with show/hide toggle
- ✅ Confirm password field
- ✅ Automatic XOR encryption before storage
- ✅ Duplicate key name detection

### PDF Password Selection
- ✅ Dialog shows saved passwords when opening PDFs
- ✅ Tap saved password to use it
- ✅ "Enter New Password" option
- ✅ Save new password checkbox
- ✅ "No Password" option for non-protected PDFs

## 🔐 Security Features

### Encryption
- XOR encryption for password values
- Encryption key stored in flutter_secure_storage
- Keys visible, values always encrypted
- Decryption only when needed

### One-Time Setup
- Encryption key setup (already implemented)
- Developer password protection for key access
- Secure storage using Android encrypted shared preferences

## 🎨 UI/UX

### Navigation Integration
- New bottom nav item: **Passwords** (key icon)
- 3-tab navigation: Documents | Passwords | Settings
- Smooth tab switching with IndexedStack

### Password List
- Card-based list design
- Key icon with themed background
- Creation date display
- Search bar at top

- Empty state with icon and helpful text

## 🐛 Bug Fixes

- Added null checks for encryption failures
- Proper error messages when encryption key not set
