# Release v0.0.27 - All Docs Screen & Enhanced Open With

## 📁 All Docs Screen (New)
- **Device File Browser**: Browse all documents on your device (not just app-managed files)
- **Quick Filters**: Filter by PDF, Word, Excel file types
- **Multi-Select Import**: Select multiple files and import to any folder
- **Single-Tap Open**: Tap to open files directly, or auto-navigate to existing location if already imported

## 🔄 Enhanced "Open With" 
- **Reliable Background Handling**: Fixed issue where "Open With" only worked first time when app was in background
- **Try-Finally Intent Guard**: Ensures intent processing flag is always reset
- **Duplicate Detection**: Shows Android notification when opened file already exists in library
- **Location Selection Popup**: Tap notification to see all locations and choose which to open

## 🔔 Smart Duplicate Notifications
- **Stay on Current Screen**: Notification popup appears on whatever tab you're currently on
- **Filename Display**: Shows actual filename (not "File") in duplicate selection sheet
- **Direct Folder Navigation**: Tap a folder location to navigate directly to Documents tab and that folder

## 🔄 Auto-Rename Conflict Resolution
- **Incrementing Suffixes**: Automatically uses `_1`, `_2`, `_3` etc. instead of manual suffix input
- **Consistent Behavior**: Same rename logic across All Docs and Document Dashboard imports
- **Removed Manual Input**: No more dialog asking for custom suffix - it's all automatic

## 🛠 Developer Tools Enhancements
- **Stream Logging**: Added logging for `getMediaStream` events
- **Intent Processing Logs**: Detailed logs for debugging Open With issues
- **Export Service Singleton**: Fixed notification access when app resumes from background

## 🐛 Bug Fixes
- Fixed "Open With" failing on subsequent attempts when app in background
- Fixed ExportQueueService context.read() failing when context is stale
- Fixed duplicate selection sheet showing "File" instead of actual filename
- Fixed folder navigation not switching to Documents tab
- Fixed duplicate popup always opening on All Docs instead of current screen
