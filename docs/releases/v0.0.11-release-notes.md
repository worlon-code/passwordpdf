# Release v0.0.11 - UI Improvements

**Release Date:** December 30, 2024  
**Build:** 11

## 🎨 New Features

### In-App PDF Viewer
- **Tap PDF files to view in-app** - No external apps needed
- Built-in controls:
  - Pinch to zoom in/out
  - Double-tap to auto-zoom
  - Swipe to navigate pages
  - Page scroll indicator and counter
- Professional viewing experience with Syncfusion PDF Viewer
- Error handling for missing/corrupted files

### File Information Screen
- **New "File Info" option** in file menu (3-dot)
- Displays comprehensive file metadata:
  - File size (formatted: B, KB, MB, GB)
  - File type and extension badge
  - Creation and modification dates
  - Full file path
  - Availability status
- Modern card-based UI design
- "Open File" button for PDFs in info screen

### Enhanced File Interactions
- **Long-press to enter move mode** workflow:
  1. Long-press any file → Enters move mode
  2. Tap additional files → Multi-select
  3. Tap Move icon → Choose destination
  4. Helpful snackbar guides the user
- PDF files: Tap to open viewer (when not in move mode)
- Non-PDF files: Only selectable via long-press

## 🔧 UI/UX Improvements

### Modern Move Dialog
- Redesigned with contemporary Material Design
- **Icon + Title header** with file count
- Clear "Select destination folder" instruction
- Maximum height constraint for better scrolling
- Improved visual hierarchy and spacing

### Fixed Menu Positioning
- **3-dot menu now appears near tapped item** (not static top position)
- Uses `RenderBox` for dynamic position calculation
- Seamless UX regardless of scroll position
- Menu items:
  - 📄 File Info (NEW)
  - ✓ Select
  - ✏️ Rename
  - 🗑️ Delete
