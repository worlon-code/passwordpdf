# Release Notes - v1.0.0-beta.1

**First Beta Release! 🎉**

This release marks a significant milestone as we move into the Beta phase. It includes all core functionality for secure PDF management, password storage, and file organization.

## New Features
- **File Info & Occurrences**: View detailed file metadata and track duplicate files across your library.
- **Import Overhaul**: Robust import flow with conflict resolution (Duplicate/Rename/Skip).
- **Duplicate Prevention**: Improved logic to prevent duplicate folder creation and file naming conflicts.
- **Open With**: Seamless integration with external apps to open PDFs directly into PasswordPDF Manager.

## Improvements
- **Folder Navigation**: Tab switching fixes for smoother navigation.
- **UI Enhancements**: Added trailing menu to "All Docs" for quick access to File Info.
- **Performance**: Optimized duplicate file checking logic.

## Bug Fixes
- Fixed "Unorganized Files" label appearing incorrectly in File Occurrences.
- Fixed navigation tab not switching when opening duplicates.
- Fixed duplicate folder creation allowing space-padded names.

## Known Issues
- None at this time.

Thank you for testing the beta! Please report any issues on GitHub.
