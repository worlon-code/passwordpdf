# Release v0.0.26 - Appearance Settings & Developer Tools

## 🎨 Appearance Settings
- **Theme Dropdown**: System / Light / Dark options
- **Accent Color Picker**: 8 preset colors + advanced HSV picker
- **HSV Color Picker**: Gradient selector, hue bar, and hex input
- **Font Size Adjustment**: Slider to reduce font size by up to 7px (default -3px)

## 📁 Cross-Folder Duplicate Check (Add Files)
- When adding files, checks ALL folders for duplicates (name + size match)
- Shows dialog listing existing file locations
- Tap file name to navigate to that folder
- "Import Anyway" proceeds with conflict resolution

## 📤 Open With Improvements
- **Loading Screen**: Shows "Opening file..." during import delay
- **Duplicate Detection**: Checks existing files across all folders
- **Android Notification**: Shows folder location for duplicates
- **Password Flow**: Password-protected PDFs now prompt correctly

## 🛠 Developer Tools
- Developer Screen with password protection
- Debug logs viewer
- Database table viewer with pagination (50/100/200/500/All)
- Export tables to Excel
- CRUD operations for records (except encryption key)

## 🐛 Bug Fixes
- Fixed font size crash when changing slider
- Fixed duplicate detection not working after loader addition
- Fixed password flow not triggering for shared PDFs
