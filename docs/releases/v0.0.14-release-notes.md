# Release v0.0.14 - Menu Positioning Fix

**Release Date:** December 30, 2024  
**Build:** 14

## 🎯 Key Fix

### Menu Positioning - FIXED ✅
- **Problem:** 3-dot menu appeared at static top position regardless of item location
- **Solution:** Replaced manual `showMenu()` with `PopupMenuButton` widget
- **Result:** Menu now automatically positions near the tapped item
- **Technical:** PopupMenuButton handles positioning internally using Flutter's automatic layout positioning

### How It Works
- Widget-based approach instead of programmatic positioning
- Flutter renders menu relative to button position
- Works correctly when scrolled down or at any position
- Standard Material Design pattern
