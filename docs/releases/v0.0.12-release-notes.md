# Release v0.0.12 - Bug Fixes

**Release Date:** December 30, 2024  
**Build:** 12

## 🐛 Bug Fixes

### Android Back Button in Move Mode
- **Fixed:** Android back now properly cancels move/selection mode instead of closing app
- Behavior hierarchy:
  1. If files selected → Clear selection
  2. If inside folder → Navigate to parent  
  3. If at root → Close app
- Improved `PopScope` logic with proper state checking

### File Storage Optimization
- **Verified:** App uses original file path references (no cache copying)
- Files remain in their original locations
- No unnecessary storage duplication
- Efficient file management

### Duplicate File Rename
- **Implemented:** Physical file copying for renamed duplicates
- When duplicate detected and "Rename & Add" chosen:
  1. User enters new name
  2. **Original file is copied** to same location with new name
  3. Copy is added to folder (original untouched)
- Proper error handling for copy failures
- Maintains file integrity
